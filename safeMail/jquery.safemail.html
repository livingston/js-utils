safeMail

Anti-Spam Email with Unobtrusive Javascript

Overview :

Email address potection from Spam bot with Unobtrusive Javascript. Read about safeMail.

Examples :

DOM
  • livingston.samuel [at] gmail [dot] com - Single Class
  • livingston [at] xatre[dot]info - Multiple Classes

Usage :

The email address must be added in the form as shown below,

<span class="safemail">livingston.samuel [at] gmail [dot] com</span>
<span class="hello safemail">livingston[at]xatre[dot]com</span>

You can use multiple class names for the span element to add your own styles.
